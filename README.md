# SEQUORR

### A community-focused fitness movement.

Sequorr is more than just a fitness app; it's a platform designed to empower individuals through community, immersive technology, and data-driven insights. Our mission is to make fitness an engaging, fluid, and high-performance experience for everyone.

---

## 🏗️ Project Overview

This repository is organized into distinct modules to ensure scalability and maintainable growth.

### 💻 [Frontend](./frontend/)
The user-facing side of Sequorr. Built with React, Vite, and Three.js, it delivers a premium, high-performance web experience with 3D graphics and smooth animations.
- **Technologies**: React 19, Three.js, Motion, Face-api.js.
- **Documentation**: [Frontend README](./frontend/README.md)

---

## 🚀 Getting Started

To explore the project locally, navigate to the module you wish to work on:

```bash
# Go to frontend
cd frontend
# Install and run
npm install
npm run dev
```

For detailed instructions on the web application, please refer to the [Frontend README](./frontend/README.md).

## 🌐 Our Vision

We believe that fitness should be:
- **Immersive**: Using 3D and high-performance visuals to engage the senses.
- **Community-Driven**: Connecting people through shared goals and progress.
- **Intelligent**: Leveraging AI and data to provide personalized fitness journeys.
